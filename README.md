## 运维

安装
npm install -g hexo


安装依赖
npm install


列出本地所有node版本

nvm list


版本切换

nvm use v15.0.0


## 博客运营

本地运行
hexo s

清理
hexo clean

部署
hexo d

### 创建文章

hexo new "post title with whitespace"

### 图片插件

npm install https://github.com/CodeFalling/hexo-asset-image -- save
